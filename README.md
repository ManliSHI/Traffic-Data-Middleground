# 哈尔滨出租车数据中台

基于哈尔滨出租车GPS轨迹数据的交通数据中台系统，实现路况监测、热点区域分析、轨迹可视化、长短单推荐及司机服务榜等功能。

## 项目概述

本项目构建了一个面向出租车行业的数据中台系统，覆盖从数据接入到价值变现的完整链路。项目基于哈尔滨市5天的出租车GPS轨迹数据（约13,000辆出租车，超过100万条轨迹）和OpenStreetMap路网数据，完成了数据汇聚整合、提纯加工、服务可视化和价值变现四个阶段的工作。

面向的主要用户角色包括：政府交通管理部门（宏观监测与规划决策）、出租车/网约车企业（运力调度与运营管理）、司机（接单策略与路线规划）以及乘客（出行参考与服务体验）。

## 四层数据体系

| 层级 | 名称 | 核心内容 | 主要产出 |
|------|------|----------|----------|
| ODS | 贴源数据层 | 原始HDF5数据接入、Barefoot地图匹配 | 5个JLD2文件 |
| DW | 统一数仓层 | 数据清洗、异常检测、时间规范化、格式统一 | 5张Parquet表 |
| TDM | 标签数据层 | 车辆/道路/时段/区域标签画像 | 7张标签表，40+个标签 |
| ADS | 应用数据层 | 业务指标聚合、图表生成 | 23张结果表，23张图表 |

| 路况监测 | 订单量趋势、高峰时段识别、道路风险排行、OD需求流向 | 订单时段分布图、道路风险排行、OD流向地图 |

## 数据说明

- **原始数据来源**：哈尔滨市出租车GPS轨迹数据集，来源于论文《Learning Travel Time Distributions with Deep Generative Model》
- **时间范围**：2015年1月3日 至 2015年1月7日（共5天）
- **数据规模**：13,000辆出租车，约1,199,531个trip，50,781,857个GPS点
- **地图数据**：OpenStreetMap（OSM）哈尔滨路网数据，14,497个道路路段

**注意**：本仓库仅包含代码、文档和预览数据（CSV格式）。完整的Parquet数据文件（约8GB）因GitHub存储限制未上传，可通过运行脚本从原始数据重新生成。
